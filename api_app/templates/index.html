007 API RESTful API for EON Production James Bond Films

Introduction

First and foremost, this API was created heavily based on the Star Wars API. If you've never visited swapi then I highly encourage you to do so.

During the week after instruction finished at the Iron Yard in Greenville, SC and with time before final projects began, I decided to build this API. I have been a life-long Bond fan and there are many others like me. It was disappointing to me in my, albeit brief, Google search that I couldn't find an existing API for James Bond information. I hope that this API may be of use to someone, and I'll continue to try and make it better. This project is fully open source and built on top of Python and Django REST Framework. Anyone who would like to contribute is completely welcome. You can find the repository here.

Please feel free to visit my personal website and view some of my other projects as well.

Documentation

All information in the database can be accessed via the root api url /api/ and appending the plural name of the resource: 

/api/characters/
This will return a paginated list of all characters; default page size is 10 records.
Resources that may be queried are: 
  • /api/characters/
  • /api/movies/
  • /api/gadgets/
  • /api/vehicles/
  • /api/bond-actors/

You may also filter down the characters by the role they play in the movies by replacing /characters/ with a more specific resource name.
Accepted resource names are: 

  • /allies/
  • /villains/
  • /henchmen/
  • /bond-girls/

Appending a query string ?page=2 will return page two of the results. Thus, the following url returns page 3 of the vehicles used in Bond films: 

/api/vehicles/?page=3

Finally, you may also retrieve specific records by appending the id of the record to any specific endpoint.

For example, using this format /api/<resource-name>/:id/ the following url 

/api/characters/1/
will return: 
      {
          "id": 1,
          "name": "James Bond",
          "bio": "Intelligence officer in the Secret Intelligence Service, commonly known as MI6. Bond is also known by his code number, 007, and is a Royal Naval Reserve Commander",
          "alias": "007",
          "ally": false,
          "villain": false,
          "henchman": false,
          "bond_girl": false
      }
the man himself: Bond, James Bond.
